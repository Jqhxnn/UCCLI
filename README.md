## UCCLI - by Jqhxnn

This is a terminal application developed in JavaScript that helps you convert units. 
Instead of using Google or AI you just execute a simple command in your terminal.

### Installation

To install and use this application follow these steps:

1. Clone the GitHub repo:
``` 
git clone https://github.com/Jqhxnn/UCCLI.git
```

2. Make the command accessable:
```
npm link
```

3. Test if the app works:
```
uccli help
```

### Usage

Currently there are following units available: km, mi, m, cm, dm, c, f

Example how to use it:
```
uccli 5 km mi
```
