## UCCLI - by Jqhxnn

This is a terminal application developed in JavaScript that helps you convert units. 
Instead of using Google or AI you just execute a simple command in your terminal.

### Installation

To install and use this CLI tool, follow these steps:

1. Install the package from GitHub Packages

```bash
npm install -g @jqhxnn/uccli
```

2. Verify the installation

```bash
uccli help
```

### Requirements

Make sure you have:

- Node.js installed

### Usage

Currently there are following units available: km, mi, m, cm, dm, c, f

After installation, you can run the CLI anywhere:

```bash
uccli help
```
